# `dolist` - Listeniteration

## Konzept

`dolist` iteriert nacheinander über die Elemente einer Liste und führt einen Block Code darauf aus.

## Beispiel

```lisp
(dolist (x '(1 2 3))
    (format t "~a " x))
;; Ausgabe: 1 2 3
```

## Rückgabewert

`dolist` gibt standardmäßig `nil` zurueck,
 es sei denn, ein alternativer Rückgabewert wird angegeben:

```lisp
(dolist (x '(1 2 3) 'done)
    (print x))
;; => DONE
```

## Merkpunkte

- einfache Listeniteration
- Bindet die SChleifenvariable nacheinander an jedes Element
- optionaler Return-Wert moeglich
