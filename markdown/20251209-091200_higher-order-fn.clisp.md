# Funktionsobjekte: `lambda`, `#'`, Higher-Order Functions

## Konzept

Lisp behandelt Funktionen als First-Class-Objekte; sie können erzeugt, gespeichert und übergeben werden.

## Beispiel: Lambda-Ausdruck

```lisp
(mapcar #'(lambda (x) (* x x))
        '(1 2 3))
;; => (1 4 9)
```

## Bedeutung von `#'`

`#'` ist `(function …)` und dient der Funktionsquote:

```lisp
#'(lambda (x) x)
;; ist äquivalent zu
(function (lambda (x) x))
```

## Closure-Beispiel

```lisp
(defun make-adder (n)
  #'(lambda (x) (+ x n)))

(funcall (make-adder 10) 5)
;; => 15
```

## Merkpunkte

- Funktionen sind erstklassige Werte  
- Lambdas erzeugen anonyme Funktionen  
- Closures „merken“ sich ihre Umgebung  
- `mapcar`, `remove-if`, `sort` usw. nutzen Functions als Argumente
