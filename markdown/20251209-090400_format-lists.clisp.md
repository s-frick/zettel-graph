## Konzept
`~{ ... ~}` iteriert automatisch ueber eien Sequenz und formatiert jedes Element gemaess dem enthaltenen Formatstring.

```lisp
(format t "~{~a ~}" '(1 2 3))
;; Ausgabe: 1 2 3
```

## Anwendung auf Key/Value-Paare

```lisp
(format t "~{~a: ~a~%~}" '(:title "Kind of Blue" :artist "Miles Davis"))
;; Ausgabe:
;; TITLE: Kind of Blue
;; ARTIST: Miles Davis
```

## Merkpunkte
- wiederholt  das Format fuer jedes Element bzw. Elementpaar
- sehr maechtig fuer strukturierte Ausgaben
- funktioniert auf allen Sequenzen
