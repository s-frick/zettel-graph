# `defvar` - globale dynamische Variable

## Konzept

`defvar` definiert eine *dynamische* globale Variable, die nur initialisiert wird, wenn sie zuvor ungebunden war.

## Beispiel
```lisp
(defvar *db* nil)
```

- definiert eine global sichtbare Variable
- dynamisch gebunden (nicht lexikalisch!)
